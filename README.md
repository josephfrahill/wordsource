word-origin

A lightweight browser extension that instantly shows the linguistic origin of a highlighted word.

✨ Features
Highlight any word to see its origin (Germanic, Latinate, etc.)
Fast keyboard lookup with Ctrl + Q
Works across most websites
Supports Google Docs (via clipboard fallback)
Simple, unobtrusive tooltip UI
⚡ Usage
Mouse: Highlight a word → origin appears automatically
Keyboard: Highlight text → press Ctrl + Q
🧠 How it works

The extension checks a local dataset of words and displays:

Origin (e.g. Germanic, Latinate)
Base form (if applicable)
Source language
⚠️ Limitations
Only supports single-word lookups
Some words may return Unknown
Google Docs requires copying (Ctrl + C) before lookup
